View all tags

      Dialog for preview of tags contains table with all previously entered tags. Each column of table contains certain atribute name and alows you to sort types in alfabet or reverse alfabet order. Below the table, you can see form for modifying one specific tag.

When you select one item from table previously entered values appears and you can change them by positioning in appropriate cell. Value "Id" is unique and onced, when new tag is made, you have no more possibility of changing. Other values like "Name", "Color" and "Description" have posibility of changing in every moment.

Click on button "Save" is saving your changes, unlike the click on button "Cancel", which turns you on previous conditions of values. By pressing on button "Delete" you can delete previously selected tag.
