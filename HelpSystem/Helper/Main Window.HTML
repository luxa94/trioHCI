Application main window

      Aplication main window contains a list view of premises, manu bar, 4 maps and 6 buttons.

List view on the right hand side shows the existing premises in sistem with their icons which picturesquely represent each of them. The list is directly related with all of 4 maps with possibility of manipulation. Once, when you select item from the list you can easily drag it to one of the maps and drop it on place you wish. In that way you can mark the place on the map where that premises is alocated.!!!!!!!!!!!!!!!!

Maps are here to help you to find premisses you are searching for. !!!!!!!!!!!!!!!!!!!!!!!!!!!!!!!

Click on button "View all premises" opens new dialog which provides you an opportunity to preview all of existing premises together with posibilitty of sorting them by atributes from table and filtering. Also, in that dialog, you are able to modify or delete selected premises.

Click on button "View all types" opens new dialog which provides you an opportunity to preview all of existing types together with posibilitty of sorting them by atributes from table and filtering. Also, in that dialog, you are able to modify or delete selected type.

Click on button "View all tags" opens new dialog which provides you an opportunity to preview all of existing tags together with posibilitty of sorting them by atributes from table and filtering. Also, in that dialog, you are able to modify or delete selected type.

Click on button "Add new premises" opens new dialog which provides you an opportunity to add new premises to list of existing ones.

Click on button "Add new type" opens new dialog which provides you an opportunity to add new type to list of existing ones.

Click on button "Add new tags" opens new dialog which provides you an opportunity to add new tags to list of existing ones.
