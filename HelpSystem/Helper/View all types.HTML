View all types

      Dialog for preview of types contains table with all previously entered types. Each column of table contains certain atribute name and alows you to sort types in alfabet or reverse alfabet order. Below the table, you can see form for modifying one specific type.

When you select one item from table previously entered values appears and you can change them by positioning in appropriate cell. Value "Id" is unique and onced, when new type is made, you have no more possibility of changing. Other values like "Name" and "Description" have posibility of changing in every moment. Also, you can browse picture from sysem by pressing on appropriate button.

Click on button "Save" is saving your changes, unlike the click on button "Cancel", which turns you on previous conditions of values. By pressing on button "Delete" you can delete previously selected type.
