View all types

      Dialog for preview of types contains table with all previously entered types. Each column of table contains certain atribute name and alows you to sort types in alfabet or reverse alfabet order. First row of table is filter row. Below the table, you can see form for modifying premises.

Filter row in the table gives you opportunity to filter all types by the certain atribute. Above each column heder of table you can see filter box where you need to enter or select value for search. After that, all types which do not match with that filter will disapeare from table and you will see just appropriate ones.

When you select item from table you can modify in form for modifying premises. Previously entered values appears and you can change them by positioning in appropriate cell. Value "Id" is unique and onced, when new type is made, you have no more possibility of changing. Other values like "Name" and "Description" have posibility of changing in every moment. Also, you can browse picture from sysem by pressing on appropriate button.

Click on button "Save" is saving your changes, unlike the click on button "Cancel", which turns you on previous conditions of values. By pressing on button "Delete" you can delete previously selected premises.
