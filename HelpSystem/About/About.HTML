About "Map your premises"

      "Map your premises" is a simple application for keeping record of geographical distribution of premises in some city. You can easily add new premises and position it on right place on one of four maps. Cities of interest in this application are Belgrade, Novi Sad, Moscow and Wroclaw, and each of maps is positioned in one of them. If you want to add premises to some other city, all you need to do is to zoom out the map of the nearest city to that one. Also, this applicaion provides you opportunity to preview all existing premises in application, to modify each one you want, or to delete one if it is no longer needed. Help system and posibility of filtering, searching and sorting premises makes the easiest way to select them. The same actions are possible for tags and types which describes premises in the system, so user can always know what tipe of premises he is looking for.
