View all tags

      Dialog for preview of tags contains table with all previously entered tags. Each column of table contains certain atribute name and alows you to sort premises in alfabet or reverse alfabet order. First row of table is filter row. Below the table, you can see form for modifying tag.

Filter row in the table gives you opportunity to filter all tags by the certain atribute. Above each column heder of table you can see filter box where you need to enter or select value for search. After that, all premises which do not match with that filter will disapeare from table and you will see just appropriate ones.

When you select item from table you can modify in form for modifying tag. Previously entered values appears and you can change them by positioning in appropriate cell. Value "Id" is unique and onced, when new tag is made, you have no more possibility of changing. Other values like "Name", "Color" and "Description" have posibility of changing in every moment.

Click on button "Save" is saving your changes, unlike the click on button "Cancel", which turns you on previous conditions of values. By pressing on button "Delete" you can delete previously selected premises.
